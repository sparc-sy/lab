## الوصف المختصر
لوحة تطوير تجمع ESP32 مع راديو LoRa (مثل SX1276/SX1278)، وشاشة OLED صغيرة أحياناً. مناسبة لشبكات LoRa طويلة المدى واستهلاك منخفض (IoT، مراقبة، تجارب حقلية).

## الاستخدامات الممكنة
- شبكات LoRa (LoRaWAN أو نقطة–نقطة)
- مراقبة عن بعد (حقول، منازل، مستشعرات)
- اتصال لاسلكي بعيد المدى باستهلاك منخفض
- عقدة IoT في مناطق ضعف التغطية
- مشاريع تعليمية لبروتوكول LoRa

## المواصفات التقنية الأساسية

| المواصفة | القيمة |
|----------|--------|
| الفولتية | 3.3 V / 5 V (حسب اللوحة) |
| المعالج | ESP32 |
| LoRa | SX1276/78 (868/915 MHz نموذجياً) |
| المدى | كيلومترات (حسب البيئة والهوائي) |
| الواجهة | SPI (LoRa)، I2C (OLED إن وُجد)، GPIO |
