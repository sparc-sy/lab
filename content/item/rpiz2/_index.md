لوحة Raspberry Pi Zero 2 W مع كيت (كابل، مصدر طاقة، إلخ). نفس معمارية Pi 3 تقريباً بحجم أصغر واستهلاك أقل.

## المواصفات التفصيلية (من الداتاشيت)

| المواصفة | القيمة |
|----------|--------|
| SoC | BCM2710A1 (Quad-core Cortex-A53 @ 1 GHz) |
| الذاكرة | 512 MB LPDDR2 |
| الجهد | 5 V |
| Wi‑Fi | 2.4 GHz 802.11n |
| Bluetooth | 4.2 / BLE |
| الواجهات | 1× micro USB (OTG), 1× micro USB (power), Mini HDMI, GPIO 40-pin (HAT) |
| الأبعاد | 65 mm × 30 mm × 5 mm |

## أمثلة الاستخدام

- تشغيل Raspberry Pi OS (Lite) وتوصيله عبر SSH عبر Wi‑Fi.
- استخدامه كعقدة Node-RED أو MQTT لـ IoT.
- تركيب كاميرا Pi وتشغيل motion أو مشروع مراقبة بسيط.
