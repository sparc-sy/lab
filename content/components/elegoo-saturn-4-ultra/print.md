## الوصف المختصر
طابعة ثلاثية الأبعاد تعمل بتقنية SLA/MSLA (الراتنج/UV). تنتج قطعاً بدقة عالية وسطح ناعم، مناسبة للنماذج الدقيقة والمجوهرات والقطع الطبية والتعليمية التي تحتاج تفاصيل دقيقة.

## الاستخدامات الممكنة
- نماذج دقيقة (minatures، مجوهرات، أسنان)
- قطع ذات تفاصيل دقيقة وثقوب صغيرة
- تعلم الطباعة بالراتنج وما بعد المعالجة (غسل، معالجة UV)
- مشاريع فنية وتصميم صناعي
- قوالب وأجزاء بصرية صغيرة

## المواصفات التقنية الأساسية

| المواصفة | القيمة |
|----------|--------|
| التقنية | MSLA (LCD + UV) |
| حجم البناء | راجع الطراز (مثلاً 228×128×260 mm أو قريب) |
| دقة الشاشة | 4K أو أعلى (حسب الطراز) |
| المواد | راتنجات UV (Standard, Tough, Flexible، إلخ) |
| الفولتية | 110–220 V AC (حسب المنطقة) |
| ما بعد الطباعة | غسل بالكحول، معالجة UV (تصلب نهائي) |
