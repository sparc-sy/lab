لوحة Raspberry Pi 5 مع كيت كامل (مصدر طاقة، اختياري: علبة، مشتريات إضافية). منصة قوية للتطوير والتعلم والمشاريع المنزلية.

## المواصفات التفصيلية (من الداتاشيت)

| المواصفة | القيمة |
|----------|--------|
| SoC | BCM2712 (Quad-core Arm Cortex-A76 2.4 GHz + VideoCore VII) |
| الذاكرة | 16 GB LPDDR4X |
| الجهد | 5 V DC (موصى: 5 A مع PSU الرسمي) |
| USB | 2× USB 3.0, 2× USB 2.0 |
| الشبكة | Gigabit Ethernet |
| GPIO | 40-pin (متوافق مع إصدارات سابقة مع تحسينات) |
| الفيديو | 2× micro-HDMI (4K) |
| التخزين | microSD، دعم NVMe عبر HAT |

## أمثلة الاستخدام

- تثبيت Raspberry Pi OS واستخدامه كحاسوب أو خادم.
- توصيل الكاميرا الرسمية واستخدام libcamera أو OpenCV.
- استخدام GPIO مع Python (RPi.GPIO أو gpiozero) لمشاريع الإلكترونيات.
